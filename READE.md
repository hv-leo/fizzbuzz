# fizzbuzz
Simple implementation of FizzBuzz following the TDD software development practice.

- Pre-requisites: 
  - [Python3](https://www.python.org/downloads/)

### Installation
```
git clone https://github.com/hv-leo/fizzbuzz
pip install -r requirements.txt
python src/main.py
```

### Run the Tests
```
pip install -r requirements.txt
pytest
```
